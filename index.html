The Subset Sum Problem

Now let us understand another important problem solved using backtracking… The Subset Sum Problem.

We are given a set of numbers: {3, 4, 5, 2}
And a target value: 7

Our goal is to find a subset whose sum equals the target.

At each step, we have two choices:
• Include the element
• Exclude the element
